# Final Web App 🌊⌨️

A React-based web application that combines interactive typing experiences with dynamically generated content using free public APIs. This project was developed for a Web Application Development course and focuses on front-end development, API integration, responsive design, and creating an engaging user experience.

## 🌟 Project Overview

This application allows users to complete typing challenges while displaying randomly generated quotes and visuals pulled from free APIs. The project was designed to create a more personalized and interactive experience by combining real-time data with responsive web design.

The app focuses on both functionality and aesthetics, featuring a clean modern layout, smooth navigation, and dynamic content generation through external APIs.

## 💻 Technologies Used

* React
* JavaScript
* HTML5
* CSS3
* Public APIs
* Responsive Web Design
* GitHub

## 🔌 API Integration

One of the main focuses of this project was integrating free public APIs into a React application.

The APIs were used to:

* Generate random quotes
* Display dynamic images/backgrounds
* Create a more interactive and unique user experience for each session

This project helped strengthen skills related to:

* Fetching API data
* Working with asynchronous JavaScript
* Handling JSON responses
* Updating React components dynamically
* Managing state and rendering external content

## 📄 Website Features

### 🏠 Home Page

Introduces the application and allows users to begin the typing experience through a clean and visually engaging interface.

### ⌨️ Typing Experience

Users complete typing challenges using dynamically generated quote content pulled from APIs in real time.

### 🖼️ Dynamic Visual Content

Images and visual elements are generated through external APIs to make the experience feel more interactive and personalized.

### 📊 Results & Feedback

Users receive feedback on their typing performance and overall experience.
